—————— Polski ——————

Twoim zadaniem będzie stworzenie aplikacji do zarządzania kolekcją zdjęć.

Wymagania:
- aplikacja musi zawierać: nawigację z przyciskami akcji, główne okno z kolekcją zdjęć
- przyciski w nawigacji powinny posiadać liczniki elementów (wszystkich, zaznaczonych) 
- przycisk "show selected" powinien pokazać zaznaczone zdjęcia
- przycisk "reset selection" powinien odznaczyć wszystkie zaznaczone zdjęcia
- przycisk "select all" powinien zaznaczyć wszystkie zdjęcia
- checkbox na pojedynczym zdjęciu powinien pokazywać się po najechaniu na element i zmieniać swój stan po kliknięciu (tak jak w projekcie)

Zalecenia:
- zwróć uwagę na właściwe użycie elementów HTML oraz styli CSS
- zadbaj o odpowiednie nazewnictwo klas zgodnie z przyjętymi konwencjami
- postaraj się stworzyć strukturę aplikacji w taki sposób, aby później łatwo było osadzić ją w innej aplikacji
- aplikacja powinna składać się z komponentów, które można ponownie użyć 

Masz jakieś pytania? Pisz śmiało!

Powodzenia!




—————— English ——————

Your task is to implement application for managing images collection.

Requirements:
- application should contain elements: navigation with action buttons, main frame with images collection
- buttons in navigation should show counters (all, selected)
- button "show selected" should show selected images
- button "reset selection" should unselect all images
- button "select all" should select all images 
- checkbox in a single image should be shown on hover and change it’s state after click (as show in design file)

Recommendations:
- pay attention to proper usage of HTML elements and CSS styles
- look to appropriate class naming according to known conventions
- try to code application in such a way that it can be embedded in other application
- apply component architecture, elements should be reusable

Any questions? Feel free to contact us!

Good luck!
